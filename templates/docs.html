    Documentation

    In this section you will find the parameters which can be used to construct a query. More coming soon

    • Name

      Accepted formats: name:VALUE n:VALUE
      VALUEtakes in english alphabets. For values with multiple words use _to separate them

    • Division

      Accepted formats: div:VALUE d:VALUE
      VALUEtakes in roman numberals and english alphabets. For values with multiple words use _to separate them

    • Date of Birth

      Accepted formats: dob:dd/mm/yyyy
      Use nto leave any of ddmmyyyyout of query [example: dob:n/1/2000 to query month Jan and year 2000]
